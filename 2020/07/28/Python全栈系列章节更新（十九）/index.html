2.多版字体确认加密字体

在开发者工具中选择Network筛选font后刷新网页就能找到这个加密字体了

如何浏览ttf格式文件的字体内容呢？

将下载好的字体文件.ttf，.woff统一转换为 .ttf 格式。

  • woff convert ttf 工具：https://cloudconvert.com/woff-to-ttf
