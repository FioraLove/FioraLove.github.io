python文件读read()、readline()、readlines()对比

AhriLove

2019-11-07

Python

   read():读取整个文件，将文件内容放到一个字符串变量中
   readline():每次读取一整行，包括换行符，并将其作为字符串返回
   readlines():读取所有行然后把它们作为一个字符串型列表返回（即把每一行数据包括换行符作为一个列表元素）
