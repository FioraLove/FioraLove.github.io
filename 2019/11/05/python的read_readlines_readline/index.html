陈浩东

2019-11-05

Python / read、readline、readlines对比

   read():读取整个文件，将文件内容放到一个字符串变量中

   readline():读取整行，包括行结束符，并以其作为字符串返回

   readline():读取所有行然后把它们作为一个字符串列表返回（即把每一行作为一个列表元素）
